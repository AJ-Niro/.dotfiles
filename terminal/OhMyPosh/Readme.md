## Pre-install

### Fonts
* Install [Hack Nerd Font](https://www.nerdfonts.com/font-downloads) in `~/.local/share/fonts`

## Install

Add the next lines to `.zshrc` file

* `export PATH=$PATH:/home/andres/.local/bin`

* `eval "$(oh-my-posh init zsh --config '~/.dotfiles/terminal/OhMyPosh/ohmyposh.json')"`
